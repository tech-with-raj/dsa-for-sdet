This repository contains Data Structures and Algorithm problems 
implemented in Java for SDET interviews.

## Purpose
To strengthen problem-solving skills required for:
- Coding rounds
- Logic-based automation challenges
- Optimization discussions
- Time and space complexity analysis

## Topics Covered
- Arrays
- Strings
- Hashing
- Sliding Window
- Two Pointer Technique
- Recursion
- Stack & Queue
- Sorting & Searching
- Basic Trees

## Problem Format
Each problem includes:
- Problem Statement
- Approach Explanation
- Time Complexity
- Space Complexity

## Tech Stack
- Java
- Maven

## Goal
To build strong algorithmic thinking required for SDET roles.
